# 🔮 Horoscope Website

[![Deploy on Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

## 🌐 Live Demo
[https://horoscope-website-3jkz.onrender.com/](https://horoscope-website-3jkz.onrender.com/)

## ✨ Features
- 🔮 Daily horoscope for all zodiac signs
- 📅 Tomorrow's horoscope available
- 🌙 Dark/Light theme toggle with persistence
- 🇯🇵🇬🇧 Japanese & English languages
- 📱 Responsive design for mobile
- 🎨 Modern gradient UI
- ⚡ Fast API integration

## 🛠️ Tech Stack
- Flask (Python)
- HTML5/CSS3
- JavaScript
- Gunicorn
- Deployed on Render

## 📦 Local Development

git clone https://github.com/JapaneseGerman/horoscope-website.git
